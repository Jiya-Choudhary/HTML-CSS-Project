# 🎵 Spotify Web Player Clone

A responsive front-end clone of the Spotify Web Player built using HTML and CSS. This project recreates Spotify's modern user interface, including the sidebar, music library, playlists, cards section, and music player controls.

## ✨ Features

* 🎨 Modern Spotify-inspired UI
* 📱 Responsive Layout
* 🎵 Music Player Interface
* 📚 Library & Playlist Sections
* 🖼️ Custom Album Artwork & Assets
* 🔍 Navigation Menu
* ⚡ Built with Pure HTML & CSS

## 🛠️ Tech Stack

* HTML5
* CSS3
* Font Awesome Icons
* Google Fonts (Montserrat)

## 🚀 How to Run

1. Download or clone the repository.

```bash
git clone https://github.com/Jiya-Choudhary/Spotify-Web-Player-Clone.git
```

2. Open the project folder.

3. Launch `index.html` in your browser.
